What is SVN?


SVN or subversion is a version control system that is a compelling replacement for CVS. Further information about SVN can be found at: http://subversion.tigris.org

SVN comes with nearly every Linux distribution. You find binaries for Linux, Windows, MacOSX, etc at http://subversion.tigris.org/project_packages.html.


Important commands

Checkout a repository
svn checkout [repository] [local-copy]

Update your local copy (since the repository has changed)
cd [local-copy]
svn update

Check the status of your local copy
cd [local-copy]
svn status


Commit your changes to a repository (only allowed to authors at openslam.org)
cd [local-copy]
svn commit
