Long Description
This software is a GNU GPL licenced Matlab toolbox for robot localization and mapping. It is made for research and education and independent on the type(s) of feature and type(s) of sensors/ It can import a number of data file formats from any sensor. It allows you to plug in and out your feature extraction, odometry model, data association strategy, etc. and to plug in and out your SLAM or Localization approach. It furthermore comes with a number of useful tools and functions.
